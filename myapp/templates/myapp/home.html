Welcome to Fake JSON website!



Easy to Use

Easy to use, even for developers who are new to working with JSON data. This requires a simple and user-friendly interface, with clear documentation and tutorials.

Fast and Reliable

The website generate Fake JSON data quickly. Fake Json included Posts, User, Comments. A blogging website is a most common project of every developer.

Secure and Private

The website is secure and private. It validate the data before serving. It also check the post, put, patch and delete requests before sending the response to the user.



Fake REST API for testing   See Docs

This website is a free Fake REST API that you can use whenever you need some fake data.


Fetch data


fetch('{{url}}api/posts/1')
.then(res => res.json())
.then(data => console.log(data));
